Exoplanet Visualizations

Hover to see details, click colors in legend to filter.
About the Visualization
This visualization explores various characteristics of exoplanets with respect to the method by which they were discovered. The three slides follow an interactive slideshow layout.

General: Help text on each of the screens guides the user to actions they can take. Parameters are always reset and annotations cleared when advancing to the next slide.

In the first scene, we have a scatter plot that shows the number of discoveries of exoplanets for each discovery method for each year. Some of the data for the transit method is so large that I had to use a logarithmic scale to spread the data more evenly, and a result I could not use a stacked bar or line graph because the trends and bar segments would be distorted.
Triggers: You can hover over the points to see the count of discoveries for a given year and discovery method. You can also click on the colors in the legend to trigger the filter parameter to show only points for a particular discovery method. When you do so, the annotations for other discovery methods go away and only the annotations for the selected one show. A help text banner at the top of the screen hints what the user can do.
Parameters: The filter parameter is able to show one particular discovery method at a time.
Annotations: The annotations on this slide show notable events in the history of exoplanet discovery next to the point corresponding to the relevant year and discovery method.

In the second scene a scatterplot is shown, which plots exoplanets according to their orbital period and stellar distance in parsecs. This gives a user a feel for how different discovery methods discover planets with different spatial characteristics. Logarithmic scales are used to spread out the data on both axes.
Triggers: The user can hover over individual points to get information about a specific planet's name along with the exact distance and orbital period of the planet. The user can also use the scroll wheel on the whitespace of the scatterplot and zoom in to see the structure of the points better. When you zoom it changes the x axis range and y axis range parameters. You can also pan by clicking and dragging. Again, this only works when you click between the points, not on the points themselves.
Parameters: The x axis and y axis range parameters are controlled by zooming and panning triggers that the user can perform on the scatterplot.
Annotations: Annotations did not make sense for this slide, so they are not used.

In the third slide, a bubble chart is shown, giving the user a feel for the relative mass of explanets discovered for each discovery method.
Triggers: As in the other graphs, the user is to hover over individual points to get specific details. The user is also able to filter method types by clicking on the color in the legend, just as in the previous two graphs.
Parameter: This chart has a filter parameter just as all of the other charts.
Annotations: This chart does not make use of annotations.
